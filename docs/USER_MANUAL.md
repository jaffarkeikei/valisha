# Operating Guide - Valisha Bead Printer

This guide documents how to operate the Valisha printer once built.

---

## Quick Start

### Step 1: Power On

1. Connect 12V power adapter
2. Press power button
3. Wait for display to show "Ready"

### Step 2: Load Beads

1. Open the top cover
2. Pour beads into hoppers (one color per hopper)
3. Fill to the MAX line (~500 beads per hopper)
4. Close top cover

### Step 3: Load String

1. Open side panel
2. Place string spool on holder
3. Thread string through guides
4. Pull string to threading needle
5. Close side panel

### Step 4: Create a Wristband

1. **Select Pattern**: Touch screen → Browse → Select design
2. **Measure Wrist**: Place wrist in opening → Press "Measure"
3. **Confirm**: Review size and pattern → Press "Print"
4. **Wait**: Watch the wristband being created (~3-5 minutes)
5. **Remove**: Carefully remove completed wristband

## Pattern Types

### Flags
- 50+ country flags pre-loaded
- Accurate row patterns (minimum 9 rows)
- Example: Kenya = 11 rows (3 black, 1 white, 3 red, 1 white, 3 green)

### Names
- A-Z letters supported
- 11 rows default
- Customizable text and background colors
- Maximum 8 characters

### Custom
- Describe desired pattern
- Manual design process

## Wrist Measurement

### Automatic Measurement

1. Relax your hand
2. Insert wrist into opening
3. Press "Measure" button
4. Hold still for 2 seconds
5. Size displayed on screen

### Manual Override

If automatic measurement seems incorrect:

```
Menu → Settings → Wrist Size → Manual
Enter circumference in mm
```

Typical ranges:
- Small Adult: 140-160mm
- Medium Adult: 160-180mm
- Large Adult: 180-200mm

## Maintenance

### Daily
- Check bead levels in hoppers
- Check string level
- Remove debris from wrist opening

### Weekly
- Clean bead hoppers with brush
- Wipe sensors with soft cloth
- Check for tangled string

### Monthly
- Deep clean all hoppers
- Lubricate linear rails (light machine oil)
- Check belt tension
- Update firmware if available

---

## Troubleshooting

### Problem: Beads Not Dispensing

| Check | Solution |
|-------|----------|
| Hopper empty? | Refill with beads |
| Beads jammed? | Gently shake hopper |
| Wrong bead size? | Use 6mm beads only |

### Problem: String Breaking

| Check | Solution |
|-------|----------|
| Tension too high? | Reduce tension in settings |
| Old string? | Replace with fresh elastic |
| String tangled? | Clear tangle, rethread |

### Problem: Wrist Size Incorrect

| Check | Solution |
|-------|----------|
| Wrist moved? | Hold still during measurement |
| Sensors dirty? | Clean sensors |
| Use manual entry | Enter size manually |
